# TIGER Line Roads

## What It Provides
U.S. Census geographic features including roads, boundaries, hydrography, address ranges, and legal/statistical geographies.

## API Type
Downloadable shapefiles/geodatabases and TIGERweb REST/WMS services.

## Limits and Access
Downloads are straightforward. TIGERweb service use should be polite and cached for repeated queries.

## Licensing
U.S. Census Bureau public data; generally treated as public domain U.S. government data.

## Attribution
Credit Census Bureau/TIGER where used.

## Commercial Use
Allowed.

## Offline or Cache Use
Excellent for downloaded offline datasets.

## Best Use In Journey Book
Fallback roads, boundaries, and geocoding support where OSM is weak.

## Pitfalls
Road data can lag OSM/local reality and lacks many kid-friendly POIs.
