# Public Land Open Data Sources

## What It Provides
Parks, protected areas, public lands, recreation sites, trails, and agency-managed boundaries.

## API Type
Downloads and agency-specific APIs.

## Limits and Access
Varies by agency and dataset.

## Licensing
Varies. Federal data is often public-domain, but not all public-land datasets are federal or restriction-free.

## Attribution
Follow each dataset's citation instructions.

## Commercial Use
Usually possible, but verify dataset terms.

## Offline or Cache Use
Good for downloaded regional packages when terms allow.

## Best Use In Journey Book
Adventure destinations, park pages, trailheads, and public-land context.

## Pitfalls
Mixed schemas, update cycles, and attribution rules.
