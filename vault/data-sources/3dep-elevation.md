# 3DEP Elevation

## What It Provides
Elevation data including DEMs and lidar-derived products at multiple resolutions.

## API Type
Downloads, image services, and National Map products.

## Limits and Access
Large files and processing cost are the practical limits.

## Licensing
USGS states 3DEP products are free and without use restrictions.

## Attribution
Credit USGS/National Geospatial Program.

## Commercial Use
Allowed.

## Offline or Cache Use
Good if downloaded by selected region. Store derived hillshade/contours rather than raw national data.

## Best Use In Journey Book
Contours, hillshade, slope, terrain teaching pages.

## Pitfalls
DEM processing can be slow and output can bloat PDFs if not simplified.
