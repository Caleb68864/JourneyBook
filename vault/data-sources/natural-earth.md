# Natural Earth

## What It Provides
Global small-scale vector and raster map data at 1:10m, 1:50m, and 1:110m.

## API Type
Downloadable files such as shapefiles, GeoPackage, SQLite, and raster packages.

## Limits and Access
No runtime API is needed after download.

## Licensing
Public domain. Natural Earth says it is free for use in any type of project.

## Attribution
Attribution is not legally required by the public-domain terms but is courteous and useful.

## Commercial Use
Allowed.

## Offline or Cache Use
Excellent. Download once and package or cache as needed.

## Best Use In Journey Book
Overview maps, title pages, region context, and large-area locator insets.

## Pitfalls
Too coarse for local navigation pages.
