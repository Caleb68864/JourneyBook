# State County GIS Data Options

## What It Provides
Local roads, parcels, parks, schools, emergency services, bridges, trails, and landmark datasets.

## API Type
Usually ArcGIS REST, shapefile/GeoJSON downloads, Socrata, CKAN, or open-data portals.

## Limits and Access
Highly variable. Some portals have rate limits, download limits, or authentication.

## Licensing
Highly variable. Do not assume public portal means public-domain.

## Attribution
Use agency-specific attribution and citation.

## Commercial Use
Must be checked dataset by dataset.

## Offline or Cache Use
Good only after license review. Cache with metadata and source date.

## Best Use In Journey Book
Local landmark quality when national datasets are missing important features.

## Pitfalls
No consistency across counties, duplicate data, stale layers, and ambiguous licenses.
