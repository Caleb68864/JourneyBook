# OpenTopoMap

## What It Provides
Rendered topographic maps based on OpenStreetMap and elevation data, with a familiar outdoor-map look.

## API Type
Public raster tiles and source/style code for self-hosting.

## Limits and Access
Public tile availability is best-effort and may change. High-volume use should contact the project or self-host.

## Licensing
The online map is CC-BY-SA per the project repository. Data inputs carry their own terms.

## Attribution
Credit OpenTopoMap and underlying data sources such as OpenStreetMap/SRTM as required.

## Commercial Use
Commercial use of the public tiles is not a safe assumption; verify current terms and project status.

## Offline or Cache Use
Do not bulk download public tiles. Self-hosting from allowed data is the cleaner offline path.

## Best Use In Journey Book
Style inspiration and quick prototype topo basemap.

## Pitfalls
Server reliability, CC-BY-SA implications, older raster pipeline changes, and possible public-service limitations.
