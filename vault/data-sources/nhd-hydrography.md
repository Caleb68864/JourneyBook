# NHD Hydrography

## What It Provides
Rivers, streams, canals, lakes, ponds, coastline, dams, streamgages, and related water features.

## API Type
USGS downloads and services. Note that NHD was retired from maintenance as of October 1, 2023, with 3D Hydrography Program becoming the current path.

## Limits and Access
Download/service access through USGS. Prefer current 3DHP where available.

## Licensing
USGS public-domain data posture.

## Attribution
Credit USGS/National Geospatial Program.

## Commercial Use
Allowed.

## Offline or Cache Use
Good when downloaded by region.

## Best Use In Journey Book
Water landmarks and educational map-reading exercises.

## Pitfalls
NHD is no longer maintained; use current hydrography products where possible.
