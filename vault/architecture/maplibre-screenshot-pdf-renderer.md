# MapLibre Screenshot PDF Renderer

Use MapLibre in a controlled browser viewport, capture print-resolution map images or PDFs, then compose pages. Pros: style consistency between preview and export. Cons: WebGL/headless quirks and pixel-density validation. Best MVP renderer.
