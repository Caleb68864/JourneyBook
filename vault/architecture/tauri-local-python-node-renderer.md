# Tauri Local Python Node Renderer

A Tauri shell can call local Node and Python sidecars. Pros: strong installed-app feel, offline potential, local file access, and a future app-store path. Cons: packaging complexity, desktop build prerequisites, sidecar management, signing/notarization work, and more friction before the PDF engine is proven.

This is not the MVP recommendation. Treat it as the later desktop/offline distribution path after the Docker-hosted web app proves atlas generation, print geometry, and source licensing.
