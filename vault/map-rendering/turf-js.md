# Turf JS

## What It Does
GeoJSON spatial analysis helpers for buffers, distances, bbox, simplification, and intersections.

## License
MIT.

## Best Fit
Client-side geometry for preview and light route/page calculations.

## Problems
Geodesic vs planar operations must be chosen carefully.

## Install Complexity
Low.

## Windows Compatibility
Excellent.

## Tauri Compatibility
Excellent.

## Offline Capability
Offline after bundled.
