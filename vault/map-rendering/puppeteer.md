# Puppeteer

## What It Does
Controls Chrome/Firefox for screenshots and PDF export.

## License
Apache-2.0.

## Best Fit
Viable HTML-to-PDF renderer.

## Problems
Chromium-only workflows are less cross-browser than Playwright.

## Install Complexity
Medium because browsers must be installed/downloaded.

## Windows Compatibility
Good.

## Tauri Compatibility
Works as sidecar automation.

## Offline Capability
Offline after browser is installed.
