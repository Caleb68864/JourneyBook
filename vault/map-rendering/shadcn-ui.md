# Shadcn UI

shadcn/ui is the preferred component foundation for Journey Book. It pairs well with React, Vite, and Tailwind CSS while keeping components local and customizable.

Use it for app shell, forms, dialogs, buttons, tabs, settings panels, project lists, export flows, and review screens. Keep the actual printable atlas page components separate from ordinary app UI so screen controls do not leak into PDF output.
