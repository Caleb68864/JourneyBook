# MGRS Grid

MGRS is a NATO/DoD grid reference system derived from UTM/UPS. It is powerful but too advanced for the MVP. Add it later as an optional educational overlay.

Because TC 3-25.26 emphasizes military grids and coordinate reporting, MGRS should remain a planned advanced feature. It should never replace Journey Book's A1/B2 page grid; it should be an optional precision layer.
