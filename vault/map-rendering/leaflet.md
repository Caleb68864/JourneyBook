# Leaflet

## What It Does
Lightweight web map library for raster tiles, markers, GeoJSON, and simple overlays.

## License
BSD-2-Clause.

## Best Fit
Good for simple raster previews.

## Problems
Limited projection support and less suited to print-styled vector tile output.

## Install Complexity
Low.

## Windows Compatibility
Excellent.

## Tauri Compatibility
Works in Tauri webview.

## Offline Capability
Works offline with local tiles.
