# Playwright

## What It Does
Browser automation for Chromium, Firefox, and WebKit; useful for render tests, screenshots, and PDF in Chromium.

## License
Apache-2.0.

## Best Fit
Best test/render automation for atlas preview and visual QA.

## Problems
Bundled browser size and PDF behavior must be verified.

## Install Complexity
Medium.

## Windows Compatibility
Excellent.

## Tauri Compatibility
Works as sidecar tool.

## Offline Capability
Offline after browser binaries and assets are installed.
