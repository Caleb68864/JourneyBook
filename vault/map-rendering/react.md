# React

## What It Does
UI library for building the atlas builder, preview panels, settings, and print review workflow.

## License
MIT.

## Best Fit
Best UI layer for the desktop app.

## Problems
React does not solve map rendering or PDF generation by itself.

## Install Complexity
Low in a Vite/Tauri setup.

## Windows Compatibility
Excellent.

## Tauri Compatibility
Excellent inside Tauri webview.

## Offline Capability
Offline-capable once bundled.
