# Tippecanoe

## What It Does
Builds vector tilesets from GeoJSON, FlatGeobuf, or CSV.

## License
BSD-style; verify exact repository license.

## Best Fit
Good for generating local vector tiles from downloaded datasets.

## Problems
Unix-oriented tooling can be awkward on Windows.

## Install Complexity
Medium to high.

## Windows Compatibility
Best through WSL/Docker unless native build is packaged.

## Tauri Compatibility
External/sidecar build tool.

## Offline Capability
Excellent generated offline tiles.
