# Tauri

## What It Does
Desktop app framework using a system webview plus Rust backend capabilities.

## License
MIT or Apache-2.0 style licensing in the Tauri ecosystem; verify crate/package licenses.

## Best Fit
Best shell for a lightweight local atlas app.

## Problems
Rust/WebView2 setup and native packaging complexity.

## Install Complexity
Medium.

## Windows Compatibility
Requires Microsoft C++ Build Tools and WebView2 on Windows development machines.

## Tauri Compatibility
Native target.

## Offline Capability
Strong, if data and render assets are packaged locally.
