# Node.js

## What It Does
JavaScript runtime for rendering workers, Playwright automation, tile tooling, and PDF/image pipelines.

## License
MIT-style Node.js license.

## Best Fit
Good for browser automation and app tooling.

## Problems
Long-running geospatial CPU work may be better in Python/GDAL.

## Install Complexity
Low.

## Windows Compatibility
Excellent.

## Tauri Compatibility
Works as a sidecar process if packaged carefully.

## Offline Capability
Works offline with local packages and data.
