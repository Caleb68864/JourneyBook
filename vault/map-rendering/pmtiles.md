# PMTiles

## What It Does
Single-file tile archive optimized for range requests and static hosting.

## License
Open specification; verify library/tool licenses.

## Best Fit
Excellent for portable offline or serverless vector tile archives.

## Problems
Tooling ecosystem is newer than MBTiles.

## Install Complexity
Low to medium.

## Windows Compatibility
Excellent.

## Tauri Compatibility
Good as local file source with MapLibre integration.

## Offline Capability
Excellent if source terms allow offline packaging.
