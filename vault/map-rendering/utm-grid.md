# UTM Grid

UTM provides meter-based grid coordinates in zones and is included on modern USGS US Topo maps. It is useful for older kids and scout-style navigation, but page grids should remain simpler by default.

TC 3-25.26 treats UTM as a core military grid concept. Journey Book should keep its simple page grid for binder navigation, but the data model should allow an optional UTM overlay later.
