# Mapnik

## What It Does
Cartographic rendering engine historically used for OSM raster tile rendering.

## License
LGPL-style; verify exact current project license.

## Best Fit
Possible self-hosted raster tile renderer.

## Problems
Complex style/data pipeline and heavier ops.

## Install Complexity
High.

## Windows Compatibility
Possible but not pleasant.

## Tauri Compatibility
Backend-side only.

## Offline Capability
Excellent if fully self-hosted.
