# Vite

Vite is the preferred frontend build tool for Journey Book. It keeps the React app fast to develop and fits the shadcn/ui plus Tailwind CSS stack.

For MVP, use Vite to build the browser UI, atlas preview screens, and client-side PDF proof-of-concept. The Docker-hosted app can serve the Vite production build from the web/API container.
