# QGIS Automation

## What It Does
Automating QGIS layouts and processing through PyQGIS.

## License
QGIS is GPL.

## Best Fit
Powerful research/prototype renderer for professional cartographic layouts.

## Problems
GPL distribution implications and heavy install footprint.

## Install Complexity
High.

## Windows Compatibility
Works but large.

## Tauri Compatibility
Poor fit for a lightweight future Tauri package unless QGIS is treated as an external install or a power-user backend dependency.

## Offline Capability
Excellent if installed locally.
