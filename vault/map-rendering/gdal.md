# GDAL

## What It Does
Core geospatial translation and processing library for raster/vector formats, reprojection, clipping, and tiling.

## License
MIT/X-style.

## Best Fit
Best backend utility for serious geospatial processing.

## Problems
Dependency packaging and command complexity.

## Install Complexity
High.

## Windows Compatibility
Works but install/package via known distributions.

## Tauri Compatibility
Use as sidecar/backend dependency.

## Offline Capability
Excellent.
