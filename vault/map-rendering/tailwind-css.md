# Tailwind CSS

Tailwind CSS is the preferred styling system for Journey Book's React UI. It also supports shadcn/ui's component styling model.

Use Tailwind for the application interface and preview surfaces. Be cautious when using Tailwind directly for printable PDF pages: the chosen PDF rendering path must support the CSS features used, and print output must be tested at Letter size.
