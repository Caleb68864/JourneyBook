# Sharp

## What It Does
High-performance Node image processing built on libvips.

## License
Apache-2.0.

## Best Fit
Great for compositing, resizing, cropping, and assembling raster outputs.

## Problems
Not a GIS library; no CRS awareness.

## Install Complexity
Low.

## Windows Compatibility
Excellent because binaries are commonly available.

## Tauri Compatibility
Works as Node sidecar dependency.

## Offline Capability
Offline.
