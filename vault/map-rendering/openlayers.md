# OpenLayers

## What It Does
Feature-rich web mapping library with strong support for projections and OGC services.

## License
BSD-2-Clause.

## Best Fit
Good if WMS/WMTS/projection-heavy preview becomes central.

## Problems
Heavier learning curve than Leaflet and less style-compatible with MapLibre vector ecosystem.

## Install Complexity
Medium.

## Windows Compatibility
Excellent.

## Tauri Compatibility
Works in Tauri webview.

## Offline Capability
Good with local sources.
