# Shapely

## What It Does
Python geometry operations: intersection, buffers, clipping, simplification, containment.

## License
BSD-3-Clause; GEOS dependency is LGPL.

## Best Fit
Best for page cell geometry and route corridor calculations.

## Problems
No CRS transformations by itself.

## Install Complexity
Medium.

## Windows Compatibility
Good via wheels.

## Tauri Compatibility
Python sidecar.

## Offline Capability
Offline.
