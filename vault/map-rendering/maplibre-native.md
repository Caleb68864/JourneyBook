# MapLibre Native

## What It Does
Native renderer for MapLibre styles on mobile/desktop platforms.

## License
BSD-style open source; verify current repository license.

## Best Fit
Potential future native rendering path.

## Problems
More integration complexity than GL JS for the Docker-hosted MVP and probably unnecessary until a native desktop/mobile renderer is clearly needed.

## Install Complexity
High.

## Windows Compatibility
Possible but more native build work.

## Tauri Compatibility
Possible future Tauri/native path, but not the simplest desktop packaging route.

## Offline Capability
Strong with local assets.
