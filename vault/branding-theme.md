# Branding Theme

Journey Book should feel like a field guide meets a national park junior ranger packet: rugged, trustworthy, outdoorsy, and warm enough for a parent to hand to a kid. It should be kid-friendly, but not childish.

## Design Phrase

Rugged field-guide clarity with junior-explorer warmth.

Alternate product phrase: a printable adventure atlas for young explorers and serious map readers.

## Visual Mood

The mental image is a kid's binder sitting next to a compass, pencil, water bottle, and folded topo map on a picnic table.

The product should feel:
- Capable.
- Outdoorsy.
- Calm.
- Hands-on.
- Slightly vintage.
- Parent-trustworthy.
- Explorer-friendly.
- Not babyish.
- Not tactical cosplay.

## Palette Direction

Use a restrained outdoor palette:
- Deep forest green as the primary brand color.
- Dark olive or moss for secondary surfaces.
- Warm bark brown for borders, rules, and map furniture.
- Tan/parchment for printable page backgrounds.
- Cream/off-white for UI panels.
- Campfire orange or trail-marker yellow as a sparing accent color.
- Black/charcoal for serious map text and printed clarity.

The palette can be earthy and rugged, but it should not become muddy. The accent color should help route lines, active controls, and challenge highlights stand out.

## Typography Direction

Use a split typography system:
- Stencil/display font for the brand, page titles, atlas labels, challenge headers, and stamped UI moments.
- Clear humanist sans for app UI and instructions.
- Highly readable print/map font for generated atlas labels, legends, and worksheets.

The stencil font should feel like trail crates, park signage, field gear, or survey markings. It should not feel military-industrial, distressed, or like a tactical video game menu. The vibe is camp box stencil, not war room.

## UI Direction

For the app:
- Use shadcn/ui components themed with earthy tokens.
- Keep controls compact, practical, and clean.
- Use icons for map, compass, printer, route, layers, grid, landmark, download, and settings.
- Buttons can hint at trail markers or stamped labels, but should remain familiar controls.
- Panels can later use subtle paper/card texture, but MVP should stay clean.
- Avoid heavy faux leather, grunge, scrapbook styling, or bubbly kid UI.

## Printable Atlas Direction

For PDFs:
- Cover page should feel like a field notebook.
- Page IDs can use stencil type: A1, B2, etc.
- Continuation labels can read like map margin notices: CONTINUE NORTH TO A2.
- Challenge pages should feel like junior ranger worksheets with checkboxes, short blanks, and simple field tasks.
- Legend and marginal information should remain serious and clear.
- Attribution, scale, compass rose, and grid locator should look like real map furniture, not decoration.

Example cover structure:
- JOURNEY BOOK.
- Adventure title.
- Explorer name.
- Date.
- Atlas pages.
- Challenge checklist.

## Guardrails

Do not make the product look like military software. TC 3-25.26 should influence map literacy and structure, not the emotional brand.

Do not make the product look childish. The kid-friendly quality should come from approachable tasks, clear visual hierarchy, and adventure language.

Do not let decorative style reduce print clarity. Printed map pages need contrast, legible labels, reliable page IDs, and visible attribution.
