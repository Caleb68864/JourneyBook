# Progressive Map Skills Curriculum

This is the tiered Level 1–4 learning curve for Journey Book pages. It fuses three sources: how schools teach map skills by age, the orienteering color progression ([[Orienteering Course Levels As Scaffold]]), and the concepts in [[TC 3-25.26 Part 1 Map Reading and Land Navigation]]. The design rule is **one new concept per level**, so a kid is never handed "a different language" all at once — the worry raised in [[Captain Input — Road Atlas vs Land Nav]].

## How Map Skills Develop By Age (the evidence)

- **Ages ~5–8 (K–2):** cardinal directions via a compass rose; simple symbols + a map key; relate a map to a familiar space; "you are here." Concrete, hands-on, local.
- **Ages ~9–12 (Gr 3–6):** intermediate directions (NE/SE/NW/SW) *after* the 4 cardinals are mastered; scale and distance; reading a coarse grid; introduction to latitude/longitude and GPS coordinates on familiar ground.
- **Teen (Gr 7+):** convert/translate coordinate formats; work in metric grids; handle declination, azimuths, and contour/terrain reasoning.

This age ladder lines up almost exactly with the orienteering White→Yellow→Orange→Green progression and with TC 3-25.26's chapter order (marginal info → grids → scale/distance → elevation/contours → orientation → azimuths/declination).

## The Four Levels

### Level 1 — Road-Atlas Grid (ages ~5–8, TD1, orienteering White)
*Concept added: "find our spot" + "how much longer."*
- Familiar road-atlas page: friendly **alphanumeric in-page grid (A1/B2)**, place-name index, recognizable landmarks, route line, continuation arrows. See [[Road Atlas Conventions]].
- Tasks: find our town/square; trace the route; estimate distance remaining from mile markers / a distance table.
- TC 3-25.26 tie: **marginal information** (page title, legend/key) in its gentlest form.
- **This is the MVP default.** No coordinates, no compass, no metric grid.

### Level 2 — Scale Bar + Cardinal/Compass (ages ~7–10, TD2–TD3, Yellow/Orange)
*Concept added: direction and measured distance.*
- Adds a **measured scale bar**, a **compass rose / cardinal directions**, and "which way is north."
- Tasks: which direction are we heading; how far is it really (use the scale bar); point to north.
- TC 3-25.26 tie: **scale and distance**, **directions/orientation** (intro to the three norths, lightly).

### Level 3 — UTM / USNG Grid (ages ~9–12, TD3–TD4, Light Green)
*Concept added: a real, georeferenced metric grid.*
- Adds a **1000 m UTM/USNG grid** with edge labels; introduces **contours/terrain** for hiking pages. Grid is **self-generated and labeled USNG** (civilian-friendly, identical to MGRS math) — see [[MGRS USNG Map Acquisition]].
- Tasks: "read right, then up" to name a grid square; give a 4-digit (1 km) then 6-digit (100 m) grid; read a hill from contour lines.
- TC 3-25.26 tie: **grids** (UTM), **elevation and relief/contours**.

### Level 4 — Full MGRS + Azimuth/Declination (teen, TD5, Green/Blue)
*Concept added: full military land-nav literacy.*
- Full **MGRS** labeling (Grid Zone Designator + 100 km square ID + 8-digit coordinate), **grid/magnetic azimuths**, **declination diagram and conversion**, distance-azimuth worksheets, terrain association.
- Tasks: shoot and convert an azimuth; plot a full MGRS grid; G-M angle conversion; plan a leg by distance + azimuth.
- TC 3-25.26 tie: the full doctrine — **grids, azimuths, declination, terrain navigation** — the advanced template already named in **Stage 6B** of [[Staged Build Roadmap]].

## Design Principle

Each level is **additive page furniture on the same engine**, not a different product: a Level 4 page is a Level 1 page with three more toggles turned on. This keeps the build cost low (it rides the existing page-furniture contract from Stage 1D) and lets one family book hold mixed-level pages so siblings of different ages share the same atlas. The recommendation for which level is the default and how to ship the tiers is in [[Land Nav Learning Curve Recommendation]].
