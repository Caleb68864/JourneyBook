# Terrain Association

Terrain association means matching map shapes to real-world landforms. For kids, start with rivers, hills, valleys, lakes, and bridges before formal contour navigation.

TC 3-25.26 points toward terrain association as a later skill after map reading, grids, scale, direction, and relief. Journey Book should sequence it the same way: first identify visible landmarks, then connect contour/relief patterns to real terrain.
