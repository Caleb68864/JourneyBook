# Overview Map Plus Detailed Page Grid

The overview page is the reader's table map. It should show the whole selected extent, the detailed page grid, page IDs, major route line, and major landmarks.

Detailed pages should include a small locator inset that highlights the current page inside the full grid. This helps kids understand where a sheet belongs without flipping back constantly.

Natural Earth is useful for very broad overview pages; USGS or OSM-derived data is better for local overviews.
