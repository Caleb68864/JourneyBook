# Map Scale

Map scale connects paper distance to real distance. Every detailed page should include a scale bar; worksheets can ask kids to estimate the distance between two landmarks and compare it with route mileage.

TC 3-25.26 reinforces two scale ideas Journey Book should support: representative fraction and graphic/bar scale. The app should print both when possible, because the bar scale remains usable if a page is resized imperfectly.
