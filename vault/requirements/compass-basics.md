# Compass Basics

A compass lesson should focus on orienting the map, reading cardinal directions, and following simple bearings. Keep true north and magnetic north separate in language. For young users, first teach that the map's top is north and the compass needle points toward magnetic north.

For the advanced template, TC 3-25.26 suggests making the three norths explicit: true north, magnetic north, and grid north. Journey Book should be able to show a declination note and explain whether an angle is a grid azimuth or magnetic azimuth.
