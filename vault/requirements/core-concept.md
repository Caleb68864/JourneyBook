# Core Concept

Journey Book turns a place into a small paper navigation experience. The app should feel closer to a field notebook or road atlas than to a GIS export panel.

The core workflow is: choose a location/route/bounding box, pick print settings, preview an atlas grid, generate pages, and print. Each page must be usable on its own and as part of a binder sequence.

The kid-friendly angle changes priorities. The map should highlight recognizable landmarks, roads, rivers, parks, schools, bridges, railroads, and simple challenges. It should avoid professional clutter unless the feature teaches a map skill.
