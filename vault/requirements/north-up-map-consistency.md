# North-Up Map Consistency

Every detailed atlas page should be rendered north-up. This keeps sheet-to-sheet continuation predictable and lets kids compare the paper map to a compass.

Route direction can be shown with arrows, but the map itself should not rotate to route heading. Rotated maps are useful for turn-by-turn driving, but they break atlas learning and make neighboring pages harder to understand.

If a decorative compass rose is included, it should indicate true north. Magnetic declination belongs in a small note or optional lesson, not as a rotated map frame.
