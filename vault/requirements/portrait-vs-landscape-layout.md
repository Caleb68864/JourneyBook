# Portrait vs Landscape Layout

Portrait is a strong default for binder-ready pages because it matches typical page protectors and school worksheets. It also leaves familiar top/bottom spaces for title and activity prompts.

Landscape can reduce page count for wide routes or east/west corridors. It may be better for road-trip atlases, but awkward in a portrait binder unless pages are rotated.

The app should let the user choose, then show the page grid count before export. Automatic orientation can be offered later by minimizing total pages or minimizing awkward blank area.
