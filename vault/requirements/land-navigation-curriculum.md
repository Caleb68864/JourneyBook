# Land Navigation Curriculum

Journey Book should teach in small practical bites: cardinal directions, map orientation, scale, distance estimation, landmarks, route following, and terrain cues.

Start with road-trip navigation before advanced terrain navigation. Kids can learn "north/east/south/west", "find the bridge", "which page comes next", and "estimate how far to the park" before learning UTM or contour interpretation.

Advanced features such as declination and MGRS should be optional lessons, not default clutter on every map.

TC 3-25.26 should guide the long-term progression: map basics, marginal information, grids, scale/distance, directions/azimuths, overlays, compass work, and relief/terrain. The MVP can be gentle, but the underlying map model should be serious enough that an experienced land-navigation instructor recognizes the structure.
