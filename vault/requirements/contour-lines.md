# Contour Lines

Contour lines teach elevation shape. They can be overwhelming on a kid map, so use them lightly, with a legend and optional challenge pages. USGS topo symbols are the best teaching reference.

TC 3-25.26 distinguishes index contours, intermediate contours, contour intervals, slopes, and terrain features. The advanced map template should preserve those ideas; the beginner template should simplify the visual load.
