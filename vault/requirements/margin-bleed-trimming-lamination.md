# Margin Bleed Trimming Lamination

Most home printers cannot print full-bleed Letter pages. Journey Book should design inside a safe printable rectangle rather than relying on edge-to-edge printing.

Lamination and trimming can remove edge labels, so continuation arrows and attribution should sit inside the safe area. Optional trim marks can be added later, but should never be required for the atlas to work.

If the user wants laminated navigation cards, offer a larger margin and slightly heavier strokes.
