# Legend Generation

Legends should explain only the symbols visible in the atlas. A kid map legend should prioritize roads, water, parks, route line, page grid, selected landmarks, and user-added important locations.

TC 3-25.26 treats marginal information and symbols as the starting point for reading a map. Journey Book's printed pages should keep a clear legend/marginal information habit even when the visual style is kid-friendly.

If the main atlas uses "see page X" callouts for important locations, the legend or index should explain that those labels point to dedicated location pages.
