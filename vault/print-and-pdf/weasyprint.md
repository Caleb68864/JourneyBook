# WeasyPrint

## What It Does
HTML/CSS to PDF renderer focused on paged media.

## License
BSD-3-Clause.

## Best Fit
Good for worksheet and document-like pages.

## Problems
Map/WebGL capture is not its strength; native dependencies can matter.

## Install Complexity
Medium.

## Windows Compatibility
Good with dependencies.

## Tauri Compatibility
Python sidecar.

## Offline Capability
Offline.
