# ReportLab

## What It Does
Python PDF generation toolkit with precise page drawing.

## License
BSD-style open-source toolkit; ReportLab also sells commercial products.

## Best Fit
Best deterministic PDF composition fallback if browser output is imprecise.

## Problems
Manual layout and map image/vector ingestion required.

## Install Complexity
Medium.

## Windows Compatibility
Good.

## Tauri Compatibility
Python sidecar.

## Offline Capability
Offline.
