# Index Page Generation

An index page should list page IDs, major landmarks, important locations, routes, and challenges. For MVP, it can be generated from the page grid and saved project records.

Important locations should get their own index section showing the saved place name, the main atlas grid page where it appears, and the dedicated location page if one was generated.
