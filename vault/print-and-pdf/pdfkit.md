# PDFKit

## What It Does
Node/browser PDF generation library with drawing APIs for text, paths, images, and pages.

## License
MIT.

## Best Fit
Good if the team wants programmatic Node PDF composition.

## Problems
Manual layout work; not a map renderer.

## Install Complexity
Low.

## Windows Compatibility
Excellent.

## Tauri Compatibility
Works as Node sidecar.

## Offline Capability
Offline.
