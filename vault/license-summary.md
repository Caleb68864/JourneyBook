# License Summary

## Cleanest Path
Use public-domain USGS/Natural Earth/Census data wherever possible, and use OSM data with correct ODbL attribution and database-share obligations understood.

## Important Distinction
OpenStreetMap data is not the same thing as OpenStreetMap-hosted raster tiles. OSM data can be used commercially under ODbL, but `tile.openstreetmap.org` has a separate service policy that forbids bulk/offline tile downloads.

## Attribution Footer Pattern
Use a per-layer footer, for example: "Map data (c) OpenStreetMap contributors, ODbL. USGS data available from U.S. Geological Survey, National Geospatial Program. Natural Earth public domain." Adjust to the actual sources on the page.

## Avoid
Do not use Google Maps content in generated printable atlases. Google's FAQ says generated electronic or printed documents cannot include Google Maps Platform data or imagery.
