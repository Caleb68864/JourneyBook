# Project Overview

Journey Book is not just a map export tool. It is a printable adventure book that turns places, routes, or regions into pages a kid can navigate with a parent, a compass, and simple reference labels.

The software should accept locations, bounding boxes, and eventually route corridors. It should compute an atlas extent, divide it into printable sheets, render each sheet north-up, and add page-to-page navigation labels so the paper book behaves like a small road atlas.

The product should stay friendly: landmarks, simple challenges, compass practice, and clear labels matter more than dense professional GIS output. The print engine must still be precise: scale, page overlap, attribution, and margins need to be treated as first-class requirements.
