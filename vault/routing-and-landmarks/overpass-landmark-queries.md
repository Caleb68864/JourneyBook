# Overpass Landmark Queries

Use Overpass to find OSM features by tag inside the atlas extent or near the route. Query examples should target durable landmarks such as `amenity=school`, `amenity=place_of_worship`, `leisure=park`, `waterway=*`, `railway=*`, and `man_made=water_tower`.
