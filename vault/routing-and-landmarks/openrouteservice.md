# OpenRouteService

OpenRouteService provides hosted API endpoints and open-source backend options. Official restrictions include 50 direction waypoints, 3,500 matrix origin-destination pairs per request, and optimization limits such as 50 routes and 3 vehicles. Use hosted ORS for prototypes and document quota fallback paths.
