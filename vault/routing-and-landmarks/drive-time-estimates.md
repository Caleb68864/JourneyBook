# Drive Time Estimates

Drive-time estimates are useful for road-trip activities but should be labeled as estimates. Routing engines model travel time from road network assumptions; actual family travel includes stops, signals, weather, and traffic.

See also: [[Landmark Selection]], [[Route Highlighting]]
