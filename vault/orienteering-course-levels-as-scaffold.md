# Orienteering Course Levels As Scaffold

Competitive orienteering already solved the "how do you take a total beginner to expert navigator without overwhelming them" problem. It uses a **color-coded course progression** tied to **Technical Difficulty (TD) levels TD1–TD5**. Journey Book can borrow this proven scaffold directly for its tiers — see [[Progressive Map Skills Curriculum]].

## The Color Progression

From easiest to hardest, the standard British/US color-coded sequence is:

- **White (TD1)** — very easy. All controls on paths/handrails. Designed for **6–10 year olds** and family groups. Navigation is "follow the line you're on."
- **Yellow (TD2)** — easy. Simple linear features: paths, walls, streams, fences (handrails). Mainly **under-12s** and families.
- **Orange (TD3)** — medium. Introduces **basic compass use and route choice** — leaving the handrail to cut across. For novice adults or experienced youngsters.
- **Light Green (TD3–TD4)** — for improvers; navigation difficulty rises with **simple contours and point features**.
- **Green / Blue / Brown / Black (TD5)** — technically difficult, advanced navigation; these share the same high navigational difficulty and differ mainly in **length/stamina** (Green shortest, then Blue, Brown, Black longest).
- **Red** — a long advanced course emphasizing stamina and sustained route choice.

Key insight: difficulty climbs on **two independent axes** — *navigational technique* (TD1→TD5) and *physical length/stamina* (Green→Blue→Brown→Red). Journey Book cares almost entirely about the **technique axis**.

## Why It Maps Cleanly To Journey Book

The orienteering progression is a curriculum disguised as a course catalog. Each color adds exactly one new navigational concept on top of the last, and the early levels are explicitly built for the same 6–12 age band as the project owner's daughters.

| Orienteering | New skill introduced | Journey Book tier |
| --- | --- | --- |
| White (TD1) | Follow a line; you-are-here | Level 1 road-atlas grid |
| Yellow (TD2) | Linear features / handrails, simple legend | Level 1→2 |
| Orange (TD3) | Cardinal direction, basic compass, route choice | Level 2 (scale bar + compass) |
| Light Green | Contours, point features | Level 3 (terrain + UTM grid) |
| Green/Blue (TD5) | Full technical navigation | Level 4 (MGRS + azimuth/declination) |

## Borrowable Mechanics

- **Named, ordered, color-coded levels** that a kid graduates through and a parent can pick from — low cognitive load, clear sense of progress.
- **"String-O" / very-young entry:** below White, beginner events use a *string course* — a continuous string the child literally follows from control to control. The Journey Book analog is the Level 1 "follow the highlighted route line, find our spot" page — zero coordinate literacy required.
- **One-new-concept-per-level discipline.** Never introduce contours, compass, and a grid in the same jump. This is the antidote to "Army land nav speaks a different language" from [[Captain Input — Road Atlas vs Land Nav]].
