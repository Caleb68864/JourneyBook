# Accepting Locations

Locations can be addresses, coordinates, searched places, imported CSV rows, or clicked map points. The app should normalize them into WGS84 points with labels, source metadata, and confidence.

Journey Book should support saved important locations such as Grandma's house, school, a town, campground, trailhead, or vacation stop. These user-chosen places can appear on the main atlas, in the index, and on dedicated location pages.

Saved locations should store a display name, geometry, category, source/geocoding confidence, notes, symbol choice, and page-generation options.
