# Route-Shaped Maps

Route-shaped atlases buffer a route instead of mapping a rectangle. They can reduce pages for long trips, but page sequencing and neighbor labels become harder. Defer until rectangular atlases work.

See also: [[Printable Atlas Generation]], [[Custom Page Grid]]
